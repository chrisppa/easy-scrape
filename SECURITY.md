Security Policy

Supported Versions

This project is pre-1.0; we aim to fix security issues on the latest `main` branch and may backport critical fixes when feasible.

Reporting a Vulnerability

- Please report security issues privately via GitHub Security Advisories:
  https://github.com/chrisppa/easy-scrape/security/advisories/new
- Alternatively, you can contact the maintainers via a private issue link or email once provided in repository settings.
- Please include a detailed description, reproduction steps, and potential impact.

Response

- We will acknowledge reports within 72 hours and provide an initial assessment.
- We will work with you to validate, remediate, and coordinate disclosure.
